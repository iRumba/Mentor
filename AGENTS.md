# Mentor

## What this is

A single-file knowledge base — a competency matrix for Senior .NET Backend developers. Not a code project. No packages, no build, no tests, no CI.

## Key file

- `senior-dotnet-backend-competencies.md` — the only meaningful file in the repo. Structured with 10 sections covering C#, ASP.NET Core, architecture, databases, testing, DevOps, security, etc.

## Editing rules

- Preserve the existing markdown structure and Russian language.
- When adding new competencies, mirror the bullet style and sectioning of the existing content.
- Do not introduce code, config files, or tooling — this repo contains prose only.
