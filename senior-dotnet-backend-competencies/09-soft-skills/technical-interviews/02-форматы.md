## Форматы
- **Live coding** — реальная задача на экране (не abacaba — реалистичная задача).
- **System design** — архитектурная задача (спроектировать URL shortener, чат).
- **Soft skills / Behavioural** — STAR-вопросы (Situation, Task, Action, Result).
- **Take-home task** — задание на дом (редко для Senior, трудоёмко для оценки).
