## Важно для интервью
- Чем отличается `struct` от `class`? (хранение, наследование, мутабельность, nullability)
- Когда `struct` выгоднее? (малые размеры, массивы, частые аллокации)
- Проблемы захвата struct в замыкание или async (boxing).
- `in` и `ref` параметры для struct (защитное копирование против `readonly`).
