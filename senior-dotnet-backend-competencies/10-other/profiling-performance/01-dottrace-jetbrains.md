## dotTrace (JetBrains)
- Профилировщик .NET-приложений (Visual Studio extension / standalone).
- **Timeline** — полная картина выполнения (CPU, memory, I/O).
- **Sampling** — быстрый обзор без модификации кода.
- **Tracing** — детальная трассировка по времени (но замедляет).
- **Line-by-line** — профилирование каждой строки кода.
