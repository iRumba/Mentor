# Configuration API, Options pattern, PostConfigure

## Configuration API
Иерархическая система управления настройками приложения с поддержкой множества источников и приоритетов переопределения. [Подробнее](01-configuration-api.md)

## Options pattern
Типизированный подход к конфигурации, позволяющий разделять настройки по классам и управлять их временем жизни (статичные, снимки, мониторинг). [Подробнее](02-options-pattern.md)

## PostConfigure
Механизмы окончательной настройки и валидации опций после всех этапов конфигурации, включая проверку данных при старте. [Подробнее](03-postconfigure.md)

## Важно для интервью
Глубокое понимание различий между интерфейсами IOptions, работа с секретами, внешние провайдеры конфигурации и порядок выполнения регистрации настроек. [Подробнее](04-важно-для-интервью.md)
