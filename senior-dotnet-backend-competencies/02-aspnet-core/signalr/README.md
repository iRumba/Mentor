## Что такое SignalR
Библиотека для реализации real-time функциональности, позволяющая серверу мгновенно отправлять данные подключенным клиентам. Абстрагирует механизмы транспорта, обеспечивая автоматический выбор наилучшего доступного протокола. [Подробнее](01-что-такое-signalr.md)

## Основные концепции
Ключевые механизмы взаимодействия через Hubs, управление адресацией сообщений (всем, группам, пользователям или конкретному соединению). [Подробнее](02-основные-концепции.md)

## Groups
Механизм логической группировки соединений для массовой рассылки уведомлений определенным категориям пользователей. [Подробнее](03-groups.md)

## Scaling (масштабирование)
Особенности работы SignalR в распределенных системах и использование механизмов Backplane (Redis, Azure SignalR Service) для синхронизации сообщений между узлами. [Подробнее](04-scaling-масштабирование.md)

## Важно для интервью
Глубокие вопросы по жизненному циклу соединения, процессу Negotiation, типизации хабов и интеграции с фоновыми сервисами. [Подробнее](05-важно-для-интервью.md)
