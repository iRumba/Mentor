# Rate limiting, Health checks, Middleware customisation

## Rate Limiting
Механизмы защиты сервера от перегрузки и DoS-атак путем ограничения количества запросов от одного клиента или группы за период времени. [Подробнее](01-rate-limiting.md)

## Health Checks
Система мониторинга работоспособности приложения и его зависимостей, используемая балансировщиками нагрузки и оркестраторами (например, Kubernetes) для управления трафиком. [Подробнее](02-health-checks.md)

## Middleware customisation
Создание и гибкая настройка конвейера обработки запросов (Request Pipeline) с использованием кастомных компонентов, условий ветвления и терминальных обработчиков. [Подробнее](03-middleware-customisation.md)

## Важно для интервью
Ключевые концепции защиты API, разница между различными типами проверок здоровья (Liveness/Readiness) и детали реализации кастомного Middleware. [Подробнее](04-важно-для-интервью.md)
