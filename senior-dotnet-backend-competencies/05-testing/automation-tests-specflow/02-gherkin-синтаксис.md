## Gherkin-синтаксис
- `Feature` — описание функциональности.
- `Scenario` — конкретный пример.
- `Given` — предусловие ("дано").
- `When` — действие ("когда").
- `Then` — проверка ("тогда").
- `Background` — общий контекст для всех сценариев.
- `Scenario Outline` + `Examples` — data-driven сценарии.
